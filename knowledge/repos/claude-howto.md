# claude-howto

> Guia visual com templates copy-paste — ponto de entrada para o [[everything-claude-code]]

## O que é

Guia visual e tutorial progressivo para dominar Claude Code. Cobre slash commands, memória (CLAUDE.md), skills, subagentes, hooks, MCP servers e CLI avançada — rota de 11-13h.

## Melhora o Claude?

**Sim — foco principal.**
Ensina como configurar o Claude para o contexto específico do projeto.

## Memória de agente?

**Parcial.**
Templates de subagentes precisam ser customizados para contexto BIM.

## Aplicação BIM

- Template `/clash-report` para relatório semanal de interferências
- CLAUDE.md com normas ABNT, padrões IFC, nomes de disciplinas
- Subagente de revisão de documentação técnica com checklist
- Skill de geração de RFI a partir de clash detectado
